# SentinelPilot Product Context

## Product Purpose

SentinelPilot is a local-first security alert investigation workspace for demonstrating enterprise-grade SecOps workflows without connecting to production security devices.

## Users

- Security operations analysts reviewing alerts, evidence, approvals, and reports.
- Engineering reviewers evaluating backend architecture, deterministic workflows, and frontend product quality.
- Project owners using the repo as a credible portfolio-grade security agent platform.

## Register

product

## Strategic Principles

- Local-first: backend, frontend, SQLite, sample alerts, logs, and evals run on the host machine.
- Safe by default: high-risk response actions create approval or simulated records only.
- Evidence-centered: raw telemetry and investigation timeline records stay inspectable.
- Enterprise calm: the interface should feel dense, consistent, and trustworthy rather than decorative.
- Bilingual consistency: Chinese and English UI branches must preserve the same structure, state colors, and component sizing.
