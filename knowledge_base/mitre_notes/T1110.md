# T1110: Brute Force

## Tactic
TA0006 - Credential Access

## Description
Adversaries may use brute force techniques to gain access to accounts when passwords are unknown or when password hashes are obtained.

## Detection
- Monitor authentication logs for multiple failed attempts within a short window.
- Correlate failed attempts followed by a successful login from the same source IP.
- Alert on distinct usernames targeted from a single source.

## Sub-techniques
- T1110.001: Password Guessing
- T1110.002: Password Cracking
- T1110.003: Password Spraying
- T1110.004: Credential Stuffing
