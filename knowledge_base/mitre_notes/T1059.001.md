# T1059.001: Command and Scripting Interpreter: PowerShell

## Tactic
TA0002 - Execution

## Description
Adversaries may abuse PowerShell commands and scripts for execution. PowerShell is a powerful interactive shell and scripting environment included in Windows.

## Detection
- Windows Event ID 4104 (ScriptBlock Logging): captures deobfuscated PowerShell code.
- Windows Event ID 4688 (Process Creation): detect `powershell.exe` with `-EncodedCommand`.
- AMSI (Antimalware Scan Interface) alerts for suspicious script content.

## Indicators
- Use of `-EncodedCommand` or `-ExecutionPolicy Bypass`.
- Download cradles: `IEX (New-Object Net.WebClient).DownloadString()`.
- Base64-encoded command lines exceeding typical length.
- PowerShell spawned by non-console parents (Outlook, Office, browser).

## Relevant Tools in SentinelPilot
- `search_logs(log_type=process, host=<HOST>)`: retrieve process creation events.
- `lookup_threat_intel`: check IPs and domains found in the decoded command.
